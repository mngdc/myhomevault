# MyHomeVault
MyHomeVault is a personal management system for tracking household inventory (appliances, electronics, furniture, valuables) and managing their associated maintenance schedules, warranty information, and repair logs.
It's designed to help homeowners stay organized, proactive with maintenance, and prepared for insurance claims. 

## Tech Stack
- HTML / CSS / JavaScript
- PHP
- MySQL

## My Role
Project Manager

## Note
This project requires a PHP server and MySQL 
database to run locally.
